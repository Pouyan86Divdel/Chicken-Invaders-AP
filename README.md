پویان دیودل ۴۰۴۱۳۴۱۲

وابستگی‌ها: این پروژه به صورت کامل با استفاده از کتابخانه‌های استاندارد جاوا انجام شده و نیازی به هیچ کتابخانه خارجی ندارد.

نحوه کامپایل و اجرا: برای اجرای صحیح بازی، دستورات زیر را در ترمینال در پوشه اصلی پروژه اجرا کنید:
دستور javac src/*.java برای کامپایل
دستور java -cp src Main برای اجرای بازی

لیست کنترل‌ها:
W / ↑ | حرکت به بالا
S / ↓ | حرکت به پایین
A / ← |‌ حرکت به چپ
D / → | حرکت به راست
Space | شلیک گلوله
P | توقف / ادامه بازی
Esc | پایان بازی و بازگشت به منوی اصلی

توضیحات پایگاه داده: برای ذخیره اطلاعات کاربران و تاریخچه بازی از sqlite استفاده شده است. این فایل‌ها به صورت خودکار در اولین اجرای برنامه ایجاد می‌شوند.

۱. فایل game.db: اطلاعات حساب‌های کاربری و تنظیمات آن‌ها در این فایل ذخیره می‌شود. ستون‌های آن شامل: نام کاربری، رمز عبور، سکه‌های جمع‌آوری شده، آخرین سطح ارتقای سفینه، بالاترین امتیاز کسب شده، آخرین مرحله رسیده و تنظیمات صدا است.
